## GmsCore Support

**GmsCore Support** is a feature/fix patch that allows you to log in using **Google Sign-In**, which normally does not work if **microG** is missing.

### How to use

1. Download **microG (GmsCore)** from here:  
   [**ReVanced GmsCore v0.3.1.4.240913**](https://github.com/revanced/gmscore/releases/tag/v0.3.1.4.240913)

2. Choose the correct variant:
   - **Default** → for most devices  
   - **Huawei** → if you’re using a Huawei device

3. Install microG and **add your Google account** inside the microG app.

4. Open your **patched Perfect Piano** app.

5. Tap **Login with Google** — and you’re done ✨
