# Perfect Patches

A repository for distributing patches for **Perfect Piano**.

These patches are designed to work with **PerfectPatcher**.
To use them, make sure you install PerfectPatcher from [here](https://github.com/luoshenshi/Perfect-Patcher).

If you’re curious about how the patches work or want to learn more, check out the **Patch Directory**—each patch has its own folder in the code section above.
